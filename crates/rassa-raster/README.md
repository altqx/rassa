# rassa-raster

Glyph rasterization helpers for the [rassa](https://github.com/altqx/rassa)
subtitle rendering pipeline.

## Fork notice

`rassa-raster` is a fork of
[`crossfont`](https://github.com/alacritty/crossfont), the cross-platform font
loading and glyph rasterization library from the Alacritty project. The upstream
rasterizer is being vendored and adapted here as the fork progresses.
